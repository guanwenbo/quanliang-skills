> 不是推销饮料，而是贩卖一种「状态」

---

| 创意 | 主题 | 情绪核心 | 目标人群 |

| **A** | 状态重启 | 疲惫→清醒的爽感 | 996上班族 |

## 🔄 创意 A：「状态重启」

### 主题概念
**当你感觉被生活掏空，3秒满血复活**

不是推销饮料，而是贩卖「瞬间清醒」的爽感。产品只是重启状态的开关。

### 情绪曲线
疲惫迷茫 → 疑惑 → 惊喜 → 满血复活

---

### A1 | 0-2s | 状态低谷（情绪铺垫）

| 项目 | 内容 |
|------|------|
| **画面** | 女生盯着电脑屏幕发呆，眼神空洞，周围一切模糊虚化 |
| **镜头** | 第一人称视角，屏幕反光映在脸上 |
| **氛围** | 灰暗、疲惫、被抽空的无力感 |
| **AI提示词** | "年轻女生坐在办公桌前，盯着电脑屏幕发呆，眼神空洞疲惫，周围背景模糊虚化，冷色调，情绪低谷，竖屏特写，电影感氛围" |
| **文案/字幕** | （无字幕，用氛围代入） |
| **音效** | 低频嗡鸣，压抑感 |

**核心**：先让用户共情「我也这样」

---

### A2 | 2-4s | 转机出现（好奇钩子）

| 项目 | 内容 |
|------|------|
| **画面** | 余光瞥见桌角的粉色饮料，像一束光 |
| **镜头** | 慢动作特写，背景虚化，瓶身反光 |
| **氛围** | 希望感开始出现 |
| **AI提示词** | "桌角放着一瓶粉色饮料，阳光透过窗户照在瓶身，形成光斑，周围虚化，温暖色调，竖屏特写，希望感氛围" |
| **文案/字幕** | "试试这个？"（手写体弹出） |
| **音效** | 环境音渐弱，清脆提示音 |

**核心**：不说产品，只说「转机」

---

### A3 | 4-8s | 重启瞬间（情绪高潮）

| 项目 | 内容 |
|------|------|
| **画面** | 喝下的瞬间，画面从灰暗变鲜艳，眼神恢复光彩 |
| **镜头** | 快速切换 + 调色变化，面部特写 |
| **氛围** | 世界突然清晰了 |
| **AI提示词** | "年轻女生喝饮料瞬间，闭眼享受，背景从灰暗渐变到明亮鲜艳，眼神恢复光彩，竖屏特写，情绪转变，电影感调色" |
| **文案/字幕** | "嗯..."（满足的长音） |
| **音效** | 清脆气泡声 + 音乐渐起 |

**核心**：展示「状态切换」的爽感，不提产品名

---

### A4 | 8-12s | 满血状态（价值呈现）

| 项目 | 内容 |
|------|------|
| **画面** | 同样的场景，但整个人发光，高效敲击键盘，嘴角上扬 |
| **镜头** | 中景，工作环境变明亮 |
| **氛围** | 自信、高效、掌控感 |
| **AI提示词** | "年轻女生坐在办公桌前，自信微笑，手指快速敲击键盘，周围光线明亮，表情专注愉悦，竖屏中景，积极氛围" |
| **文案/字幕** | "我又可以了" |
| **音效** | 轻快鼓点，键盘敲击声 |

**核心**：展示「重启后」的美好状态

---

### A5 | 12-15s | 自然落版（软性植入）

| 项目 | 内容 |
|------|------|
| **画面** | 手伸向桌角的饮料，拿起来晃了晃（展示瓶身） |

| **氛围** | 自然、不做作 |

| **文案/字幕** | "状态不好的时候，记得重启一下" |

**核心**：产品只是「工具」，状态才是「价值」

## 🌸 创意 B：「小确幸时刻」

### 主题概念
**生活中那些不值钱但很治愈的瞬间**

贩卖日常仪式感。不是需要，而是想要。给自己的小奖赏。

### 情绪曲线
平淡日常 → 发现美好 → 享受当下 → 幸福感溢出

---

### B1 | 0-2s | 日常切片（代入感）

| 项目 | 内容 |
|------|------|
| **画面** | 午后阳光洒进房间，普通但温馨的居家场景 |
| **镜头** | 空镜，缓慢移动 |
| **氛围** | 慵懒、安静、时光静止 |
| **AI提示词** | "午后阳光洒进温馨房间，窗台上有绿植，光影斑驳，安静慵懒的氛围，暖色调，竖屏空镜，治愈感" |
| **文案/字幕** | （环境音，无字幕） |
| **音效** | 轻柔环境音，远处车流 |

**核心**：建立「这就是我的生活」代入感

---

### B2 | 2-5s | 打开冰箱（仪式感开始）

| 项目 | 内容 |

### B3 | 5-9s | 享受当下（ASMR感）

## 👯 创意 C：「闺蜜暗号」

### 主题概念
**只有我们才懂的默契**

贩卖专属感和社交货币。不是推荐产品，而是展示「我们的秘密」。

### 情绪曲线

### C1 | 0-2s | 神秘开场（悬念感）

| 项目 | 内容 |
|------|------|
| **画面** | 闺蜜聚会，A突然掏出两瓶粉色饮料，B眼睛一亮 |
| **镜头** | 快速切换，神秘感 |
| **氛围** | 秘密交易的感觉 |
| **AI提示词** | "两位年轻女生聚会场景，女生A神秘掏出两瓶粉色饮料，女生B眼睛一亮会意，闺蜜默契感，竖屏中景，神秘感氛围" |
| **文案/字幕** | "懂？"（眼神交流） |
| **音效** | 悬疑音效，突然停止 |

**核心**：建立「这是我们之间的暗号」专属感

## 📊 新旧对比：硬广 vs 软性

| 维度 | 旧版（硬广） | 新版（软性） |
|:---|:---|:---|
| **核心** | 产品功能 | 情绪价值 |
| **主角** | 饮料瓶 | 用户状态/感受 |
| **卖点** | 维C含量、酸甜 | 重启状态、小确幸、默契、自我犒赏 |
| **文案** | "省XX%"、"点击购买" | "我又可以了"、"给自己5分钟" |
| **结尾** | 价格+链接 | 情感升华+品牌好感 |
| **用户感受** | 被推销 | 被理解、被治愈 |
