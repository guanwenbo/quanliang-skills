## 🔄 创意 A：「状态重启」

### 主题概念
**当你感觉被生活掏空，3秒满血复活**

贩卖「瞬间清醒」的爽感，产品只是重启状态的开关。

### 完整分镜

#### A1 | 0-2s | 状态低谷（情绪铺垫）

| 项目 | 内容 |
|------|------|
| **画面** | 女生盯着电脑屏幕发呆，眼神空洞，周围模糊虚化 |
| **镜头** | 第一人称视角，屏幕反光映在脸上 |
| **氛围** | 灰暗、疲惫、被抽空的无力感 |
| **AI提示词** | `年轻女生坐在办公桌前，盯着电脑屏幕发呆，眼神空洞疲惫，周围背景模糊虚化，冷色调，情绪低谷，竖屏特写，电影感氛围` |
| **文案/字幕** | （无字幕，用氛围代入） |
| **音效** | 低频嗡鸣，压抑感 |

**核心**：先让用户共情「我也这样」

---

#### A2 | 2-4s | 转机出现（好奇钩子）

| 项目 | 内容 |
|------|------|
| **画面** | 余光瞥见桌角的粉色饮料，像一束光 |
| **镜头** | 慢动作特写，背景虚化，瓶身反光 |
| **氛围** | 希望感开始出现 |
| **AI提示词** | `桌角放着一瓶粉色饮料，阳光透过窗户照在瓶身，形成光斑，周围虚化，温暖色调，竖屏特写，希望感氛围` |
| **文案/字幕** | "试试这个？"（手写体弹出） |
| **音效** | 环境音渐弱，清脆提示音 |

**核心**：不说产品，只说「转机」

---

#### A3 | 4-8s | 重启瞬间（情绪高潮）

| 项目 | 内容 |
|------|------|
| **画面** | 喝下的瞬间，画面从灰暗变鲜艳，眼神恢复光彩 |
| **镜头** | 快速切换 + 调色变化，面部特写 |
| **氛围** | 世界突然清晰了 |
| **AI提示词** | `年轻女生喝饮料瞬间，闭眼享受，背景从灰暗渐变到明亮鲜艳，眼神恢复光彩，竖屏特写，情绪转变，电影感调色` |
| **文案/字幕** | "嗯..."（满足的长音） |
| **音效** | 清脆气泡声 + 音乐渐起 |

**核心**：展示「状态切换」的爽感，不提产品名

---

#### A4 | 8-12s | 满血状态（价值呈现）

| 项目 | 内容 |
|------|------|
| **画面** | 同样的场景，但整个人发光，高效敲击键盘，嘴角上扬 |
| **镜头** | 中景，工作环境变明亮 |
| **氛围** | 自信、高效、掌控感 |
| **AI提示词** | `年轻女生坐在办公桌前，自信微笑，手指快速敲击键盘，周围光线明亮，表情专注愉悦，竖屏中景，积极氛围` |
| **文案/字幕** | "我又可以了" |
| **音效** | 轻快鼓点，键盘敲击声 |

**核心**：展示「重启后」的美好状态

---

#### A5 | 12-15s | 自然落版（软性植入）

| 项目 | 内容 |
|------|------|
| **画面** | 手伸向桌角的饮料，拿起来晃了晃（展示瓶身） |

| **文案/字幕** | "状态不好的时候，记得重启一下" |

**核心**：产品只是「工具」，状态才是「价值」
